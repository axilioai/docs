`Key` holds the device-key constants for [`key_press`](/driver/hardware-keys).

```python
from axilio.drivers.mobile import Key

driver.key_press(Key.HOME)
```

## Constants

| Group | Constants |
|---|---|
| Navigation | `HOME`, `BACK`, `RECENTS`, `MENU` |
| Power | `POWER` |
| Volume | `VOLUME_UP`, `VOLUME_DOWN`, `MUTE` |
| Media | `PLAY_PAUSE`, `NEXT_TRACK`, `PREV_TRACK`, `STOP` |
| System | `SEARCH`, `ASSIST` |

<Note>
  `key_press` is for device keys. To type characters, use
  [`type_text`](/driver/actions) or [`type_into`](/driver/element).
</Note>
