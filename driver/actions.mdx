Once you've found something, you act on it: tap it, type into it, press and hold,
or swipe it somewhere.

```python
from axilio.platform import Client

client = Client()

with client.session("ANDROID") as driver:
    driver.find_text("Settings").tap()
    driver.find(query="the search box").type_into("axilio")
```

| Action | What it does |
|---|---|
| `tap()` | Tap it. |
| `type_into(text)` | Tap it, then type. |
| `long_press()` | Press and hold. |
| `swipe_to(other)` | Swipe from this thing to another. |

## Act on an exact spot

If you already know where to act — an `x`, `y` point on the screen — do it
directly:

```python
driver.tap({"x": 540, "y": 1180})
driver.swipe({"x": 540, "y": 1600}, {"x": 540, "y": 400})   # scroll up
driver.type_text("hello")   # type into whatever's selected
```

| Action | What it does |
|---|---|
| `tap(point)` | Tap at a point. |
| `long_press(point)` | Press and hold at a point. |
| `swipe(from, to)` | Swipe between two points. |
| `type_text(text)` | Type into whatever's currently selected. |

<Note>
  Swiping is how you scroll: swipe **up** (end higher than start) to move *down* a
  list, and the other way to move up.
</Note>
