Some buttons aren't on the screen — Home, Back, volume. Press them with
`key_press` and the `Key` names:

```python
from axilio.drivers.mobile import Key

driver.key_press(Key.HOME)   # go to the home screen
```

## The buttons

| Group | Names |
|---|---|
| Navigation | `Key.HOME`, `Key.BACK`, `Key.RECENTS`, `Key.MENU` |
| Power | `Key.POWER` |
| Volume | `Key.VOLUME_UP`, `Key.VOLUME_DOWN`, `Key.MUTE` |
| Media | `Key.PLAY_PAUSE`, `Key.NEXT_TRACK`, `Key.PREV_TRACK`, `Key.STOP` |
| System | `Key.SEARCH`, `Key.ASSIST` |

<Note>
  `key_press` is for device buttons. To [type text](/driver/type), use `type_text`
  or `type_into` instead.
</Note>
