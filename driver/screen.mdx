A `Screen` is what [`observe()`](/driver/observe) returns: a single snapshot of all
recognized text and icons. Query it locally — no round trip to the device.

```python
screen = driver.observe()
row = screen.find_text("Sign in")          # Element | None
prices = screen.find_all_text(contains="$")  # list[Element]
```

## Methods

| Method | Returns | Description |
|---|---|---|
| `find_text(text, exact=False)` | `Element \| None` | Best text match in the snapshot. |
| `find_all_text(contains=…, pattern=…)` | `list[Element]` | Every text match. |

## Fields

| Field | Type | Description |
|---|---|---|
| `texts` | `list[Element]` | Every recognized text element. |
| `icons` | `list` | Detected icons, each with `bbox`, `center`, `confidence`. |
| `width`, `height` | `int` | Screen dimensions, in pixels. |
| `captured_at` | `datetime` | When the snapshot was taken. |
| `hash` | `str` | Content hash — compare across snapshots to detect change. |
