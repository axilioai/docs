The **Mobile driver** is how you control a phone that the
[Platform](/introduction) gives you. A [session](/quickstart) hands you one.

It's a small set of **basics** — find, tap, type, swipe — that you chain together
to do anything:

```python
from axilio.platform import Client

client = Client()

with client.session("ANDROID") as driver:
    driver.find(query="the Settings icon").tap()
    driver.find_text("Search").type_into("axilio")
```

Every basic is one idea. Learn the handful below and you can drive any app.

## Chaining them together

Most steps are a **find** followed by an action:

```python
driver.find_text("Settings").tap()          # find -> tap
driver.find(query="the search box").type_into("hello")   # find -> type
driver.wait_for_text("Welcome").tap()        # wait -> tap
```

That's the whole model. When you need exact control, every action also takes raw
coordinates — see [Tap](/driver/tap) and [Swipe](/driver/swipe).
