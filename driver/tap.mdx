Tap whatever you [found](/driver/find):

```python
driver.find_text("Settings").tap()
```

## Tap and hold

`long_press` presses and holds, for example to open a menu:

```python
driver.find_text("Photo").long_press()
driver.find_text("Photo").long_press(duration_ms=1500)   # hold longer
```

## Tap an exact spot

If you already know the point (an `x`, `y` on the screen), tap it directly:

```python
driver.tap({"x": 540, "y": 1180})
driver.long_press({"x": 540, "y": 1180})
```
