Type into whatever you [found](/driver/find). `type_into` taps the field first,
then types:

```python
driver.find_text("Search").type_into("axilio")
```

## Type into the current field

If a field is already selected, type straight into it:

```python
driver.type_text("hello")
```

<Note>
  This types text. To press device buttons like Home or Back, use
  [Buttons](/driver/buttons).
</Note>
