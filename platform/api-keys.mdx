API keys authenticate the SDK. Each key belongs to your organization and starts
with `axl_`.

## Create a key

```python
key = client.api_keys.create(name="ci-bot")
print(key.key_value)   # the secret
```

<Warning>
  `key_value` is shown once, right here. Save it somewhere safe; you can't see it
  again.
</Warning>

## List keys

```python
keys = client.api_keys.list()
```

## Regenerate or delete

```python
client.api_keys.regenerate(key_id="key_123")   # new secret, the old one stops working
client.api_keys.delete(key_id="key_123")
```
