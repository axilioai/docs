Every request to Axilio needs an **API key** — a secret that identifies your
account.

## Get a key

Create one in the [dashboard](https://axilio.ai/dashboard), under
**Settings → API keys**. Keys start with `axl_`.

<Warning>
  Your key is shown once, when you create it. Save it somewhere safe and never
  commit it to your code.
</Warning>

## Use it

The easiest way is an environment variable — Axilio reads it automatically:

```bash
export AXILIO_API_KEY=axl_your_key_here
```

```python
from axilio.platform import Client

client = Client()
```

Or pass it directly:

```python
client = Client(api_key="axl_your_key_here")
```

## Options

`Client()` takes a few optional settings:

| Setting | Default | What it does |
|---|---|---|
| `api_key` | `AXILIO_API_KEY` | Your key. Required. |
| `base_url` | `https://api.axilio.ai` | Where to send requests. |
| `timeout` | `30.0` | Seconds to wait per request. |
| `max_retries` | `3` | Automatic retries on temporary failures. |

<Note>
  Each key belongs to one organization. If you're in more than one, make a
  separate key for each.
</Note>
