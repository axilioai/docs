The `/models` endpoint returns the authoritative list of vision-language models (VLMs) that Argus accepts on the `/locate` endpoint's `model` parameter. Each entry includes the model's identifier, provider, context window size, and the per-token input and output pricing you are billed at. The list is curated — Argus enforces it at request time, so passing a model ID to `/locate` that is not in this list returns a `400 Bad Request` rather than a silent fallback. Calling this endpoint before you place your first `/locate` call lets you validate model IDs programmatically and compare costs without an extra round-trip to the inference endpoint.

## Request

**`GET https://api.axilio.ai/api/v1/inference/models`**

**Authentication:** None required. This endpoint is public.

## Response

<ResponseField name="object" type="string">
  Always `"list"`. Follows the standard `/v1/models` catalog shape.
</ResponseField>

<ResponseField name="data" type="ModelInfo[]">
  Array of supported models, sorted by `id`.

  <Expandable title="ModelInfo fields">
    <ResponseField name="id" type="string">
      The model identifier to pass as `model` in a `/locate` request. For example, `"anthropic/claude-sonnet-4-5"`. This is the canonical string Argus validates against.
    </ResponseField>
    <ResponseField name="name" type="string">
      Human-readable model name suitable for display in a UI or log message.
    </ResponseField>
    <ResponseField name="owned_by" type="string">
      The model provider, for example `"anthropic"` or `"openai"`. Useful for filtering or grouping models by vendor in your own tooling.
    </ResponseField>
    <ResponseField name="type" type="string">
      Model capability type. Always `"vlm"` for the models listed here, confirming they support vision input.
    </ResponseField>
    <ResponseField name="context_window" type="integer">
      Maximum context size in tokens. Larger images and longer `texts` grounding lists consume more input tokens — check this field if you are pushing large screenshots or extensive OCR results.
    </ResponseField>
    <ResponseField name="pricing" type="ModelPricing">
      Per-token pricing at the rate billed to your account.

      <Expandable title="ModelPricing fields">
        <ResponseField name="input" type="string">
          USD price per input token, represented as a decimal string, e.g. `"0.000003"`. Multiply by `prompt_tokens` from a `/locate` response to get the input cost for that call.
        </ResponseField>
        <ResponseField name="output" type="string">
          USD price per output token, e.g. `"0.000015"`. Multiply by `completion_tokens` from a `/locate` response to get the output cost for that call.
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

## Examples

<CodeGroup>

```python SDK
import axilio
import httpx  # GET /models has no SDK wrapper today — call it directly

# No API key needed
resp = httpx.get("https://api.axilio.ai/api/v1/inference/models")
resp.raise_for_status()

models = resp.json()["data"]
for m in models:
    print(
        f"{m['id']:<45}"
        f"  ctx={m['context_window']:,}"
        f"  in={m['pricing']['input']}"
        f"  out={m['pricing']['output']}"
    )
```

```python Validate model before locate
import httpx

def get_supported_model_ids() -> set[str]:
    resp = httpx.get("https://api.axilio.ai/api/v1/inference/models")
    resp.raise_for_status()
    return {m["id"] for m in resp.json()["data"]}

supported = get_supported_model_ids()
chosen_model = "anthropic/claude-sonnet-4-5"

if chosen_model not in supported:
    raise ValueError(f"{chosen_model!r} is not supported. Choose from: {sorted(supported)}")

location = client.argus.locate(image_bytes, query="the submit button", model=chosen_model)
```

```python Cheapest model by input price
import httpx
from decimal import Decimal

resp = httpx.get("https://api.axilio.ai/api/v1/inference/models")
models = resp.json()["data"]

cheapest = min(models, key=lambda m: Decimal(m["pricing"]["input"]))
print(f"Cheapest model: {cheapest['id']}  @ {cheapest['pricing']['input']} / input token")
```

## Using Model IDs in /locate

Once you have a model `id` from this endpoint, pass it verbatim as the `model` field in your `/locate` request:

```python
location = client.argus.locate(
    image_bytes,
    query="the hamburger menu icon",
    texts=texts,
    model="anthropic/claude-haiku-3-5",   # id from /models
)
```

<Note>
  The `/locate` response always echoes the `model` field back as reported by the provider. This may differ slightly from the `id` you passed — for example, the provider may append a version suffix. Log both for full traceability.
</Note>

## Estimating Cost Per Query

Combine `/models` pricing with the token counts in `/locate` responses to compute exact per-call costs:

```python
from decimal import Decimal

def locate_cost_usd(locate_response: dict, model_pricing: dict) -> Decimal:
    input_price = Decimal(model_pricing["input"])
    output_price = Decimal(model_pricing["output"])
    return (
        input_price * locate_response["prompt_tokens"]
        + output_price * locate_response["completion_tokens"]
    )
```

Alternatively, read `cost_microdollars` directly from the `/locate` response — Argus computes this for you using the provider's reported token counts and the same rates shown in `/models`.

## No Authentication Required

`GET /models` is intentionally public. You can call it from client-side code, CI scripts, or pre-flight checks that run before your application has loaded credentials. The model list contains no sensitive information — it is a catalog of publicly available VLMs and their pricing rates.

<Tip>
  Cache the model list at startup and refresh it periodically (for example, once per hour). Argus may add or deprecate models over time — caching avoids a blocking HTTP call on every `/locate` request while still picking up new additions.
</Tip>
