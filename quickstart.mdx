In five minutes you'll get a real Android phone, read its screen, and tap
something on it.

## Before you start

- **Python 3.10 or newer**
- **An API key** — create one in the [dashboard](https://axilio.ai/dashboard).
  See [Authentication](/authentication).

## 1. Install

```bash
pip install axilio
```

## 2. Add your API key

```bash
export AXILIO_API_KEY=axl_your_key_here
```

## 3. Control a phone

```python
from axilio.platform import Client

client = Client()

# Borrow a phone. It's returned automatically when the block ends.
with client.session("ANDROID") as driver:
    # Save a picture of the screen.
    with open("screen.png", "wb") as f:
        f.write(driver.screenshot())

    # Find something on the screen and tap it.
    driver.find(query="the Settings icon").tap()
```

Run it:

```bash
python quickstart.py
```

That's it — you borrowed a real phone, looked at its screen, and tapped something
on it.
