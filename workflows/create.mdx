A workflow is just the Python you've been writing, saved on Axilio.

## Write it in the editor

The easiest way is the **code editor** in your
[dashboard](https://axilio.ai/dashboard). Write your automation, run it against a
live phone as you go, and save it when it works. It's the same code you'd write
anywhere:

```python
with client.session("ANDROID") as driver:
    driver.find(query="the search box").type_into("axilio")
    driver.find_text("Search").tap()
```

## Or create it from the SDK

You can also save a workflow straight from code:

```python
workflow = client.workflows.create_from_code(
    name="Search Axilio",
    code_source=open("automation.py").read(),
)
print(workflow.workflow_id)
```

Update its code later with `client.workflows.save_code(workflow.workflow_id,
source=...)`.
