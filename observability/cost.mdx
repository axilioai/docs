Axilio itemizes every cost, so you always know where your money is going. No
surprises.

Costs come back in **microdollars**, a millionth of a dollar. Divide by 1,000,000
for dollars.

## Per session

Each session's cost is split into device time and vision calls:

```python
from datetime import datetime, timedelta

usage = client.usage.list_sessions(
    start_date=datetime.now() - timedelta(days=7),
    end_date=datetime.now(),
)

for s in usage.sessions:
    print(s.session_id, s.total_cost_microdollars)
    print("  device:", s.cost_microdollars)
    print("  vision:", s.inference_cost_microdollars)
```

## Per vision call

Every `find(query=...)` and other vision call is itemized:

```python
inferences = client.usage.list_inferences(
    start_date=datetime.now() - timedelta(days=7),
    end_date=datetime.now(),
)

for i in inferences.inferences:
    print(i.endpoint, i.model, i.cost_microdollars, f"{i.latency_ms}ms")
```

## Totals over time

`get_metrics` rolls everything up, with a breakdown of where the money went:

```python
metrics = client.usage.get_metrics(
    start_date=datetime.now() - timedelta(days=30),
    end_date=datetime.now(),
    granularity="daily",
)

print(metrics.compute_minutes)
print(metrics.cost_by_product)
```
