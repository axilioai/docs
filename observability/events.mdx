A run's events are the step-by-step record of what your agent did: every tap, the
screens it looked at, and the decisions it made.

```python
run = client.runs.get(run_id)
events = client.runs.list_events(session_id=run.session_id, limit=100, offset=0)

for event in events.events:
    print(event)
```
