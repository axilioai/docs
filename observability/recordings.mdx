Every session is recorded as a video. Replay it in the
[dashboard](https://axilio.ai/dashboard), or get the file from the SDK.

## A run's recording

A finished run has a `video_url`:

```python
run = client.runs.get(run_id)
print(run.video_url)
```

## A session's recording

For any session, ask for its recording by session id:

```python
recording = client.phones.session_recording(session_id)

print(recording.status)   # ready, processing, ...
print(recording.url)      # a downloadable MP4
```

## Download the MP4

The `url` is a plain link to an MP4. Save it like any file:

```python
import httpx

recording = client.phones.session_recording(session_id)
data = httpx.get(recording.url).content

with open("session.mp4", "wb") as f:
    f.write(data)
```
