## How Axilio Works

Axilio is built around three primitives:

- **Devices** — real Android phones allocated from the fleet, billed per second
- **MobileDriver** — a vision-native SDK that drives each device using natural language selectors
- **Workflows** — Python scripts you author in the dashboard and run via the API

<Steps>
  <Step title="Install the SDK">
    ```bash
    pip install axilio
    ```
  </Step>
  <Step title="Set your API key">
    ```bash
    export AXILIO_API_KEY=axl_your_key_here
    ```
  </Step>
  <Step title="Allocate a device and drive it">
    ```python
    from axilio.platform import Client
    from axilio.drivers import MobileDriver

    client = Client()

    with client.mobile.session() as device:
        driver = MobileDriver.connect()
        driver.find_text("Sign in").tap()
        driver.find_text("Email").type_into("you@example.com")
    ```
  </Step>
</Steps>
