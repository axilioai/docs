`infer` reads an image: the text on it, and the icons in it.

```python
import base64
from axilio.platform import Client

client = Client()

image = base64.b64encode(open("screen.png", "rb").read()).decode()

result = client.argus.infer(image=image, inference_type="combined")
print(result.data)
```

Choose what to detect with `inference_type`:

- `"ocr"`: the text only.
- `"yolo"`: the icons only.
- `"combined"`: both.
