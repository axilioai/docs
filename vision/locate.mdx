---

`locate` finds something in an image by description, the same way
[`find(query=...)`](/driver/find) does on a live screen.

```python
import base64
from axilio.platform import Client

client = Client()

image = base64.b64encode(open("screen.png", "rb").read()).decode()

result = client.argus.locate(image=image, query="the blue Continue button")
if result.found:
    print(result.bbox)               # where it is
    print(result.confidence)
    print(result.cost_microdollars)  # what the call cost
```

## Pick a model

Pass `model=` to use a specific [vision model](/vision/models):

```python
client.argus.locate(
    image=image,
    query="the cart icon",
    model="anthropic/claude-haiku-4.5",
)
```
